lab_2_data_model.xml (исходники диаграмм) можно открыть с помощью сервиса:
https://www.drawio.com/

Документ содержит несколько страниц, переключение между которыми возможно с помощью панели вкладок
в нижней части экрана приложения

В drawio при работе с линиями полезным инструментом является сброс опорных точек (waypoints):
при выделении объекта и/или связанных с ним линий, справа на панели задач во вкладке
«Расположение» (Arrange) → «Очистить опорные точки» (Clear Waypoints)
